
# immersivevision.us: Innovación que da forma al futuro del hogar”

immersivevision es una landing page diseñada para reflejar la fusión entre la tecnología y la arquitectura moderna. El proyecto se desarrolla utilizando HTML5 para una estructura semántica sólida, CSS3 para un diseño elegante y adaptable, y JavaScript para añadir interactividad dinámica.
Además, se incorporarán animaciones suaves con CSS y GSAP, efectos scroll parallax, componentes responsive, y optimización de rendimiento y accesibilidad, logrando una experiencia visual fluida y moderna que transmita la esencia de la innovación en el mundo inmobiliario.

## Lo que se implemento

 - 🧱 1. Estructura semántica y orden del código (HTML)
 - 🎨 2. Buenas prácticas de estilo (CSS y diseño responsive)

 - ⚡ 3. Interactividad y optimización (JavaScript)

 - 🧩 4. Organización del proyecto (arquitectura de carpetas)

- 🚀 5. Optimización de rendimiento y accesibilidad
